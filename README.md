# Лабораторная работа: Простые деревья
 
Реализация задач по бинарным деревьям на Python.
 
## Задача 2 — Дерево выражений
 
Построение дерева выражения по постфиксной записи (обратная польская запись) и рекурсивное вычисление результата.
 
### Обязательная часть
 
| Функция | Описание |
|---|---|
| `build_expression_tree(tokens)` | Построить дерево по списку токенов постфиксной записи |
| `evaluate(root)` | Рекурсивно вычислить значение выражения |
 
### Вариативная часть
 
**№6 — Унарный минус**
 
Поддерживается токен `u-`, который применяет отрицание к одному операнду.
 
```
6 u- 2 * 10 +  →  ((-6) * 2) + 10  =  -2
```
